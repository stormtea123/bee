小蜜蜂报表系统:
------------

### 项目技术：

Node.js/MongoDB/，数据模型增删改、模型集合、集合监听、模版引擎，路由控制/模块化加载 

### 项目功能： 

  * 记录每个人每天的工作安排或自助学习进度， 
  * 部门人员之间可以互相查看彼此的记录
  * 同步Google数据
  * 建立用户组
  * 周报，自动汇总
  * 邮件发送
